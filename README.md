# RetroView AI：面试复盘导师

**RetroView Interview Review** —— 将零散的面试录音文本转化为结构化、专业级复盘报告，精准定位失分项并提供黄金话术迭代。

## 核心能力

- **职级动态识别**：根据 JD 工作年限自动匹配 L1/L2/L3 评价模型
- **结构化 Q&A 提取**：智能脱水、编号、疑点标注
- **五维加权评分**：自我介绍、Q&A、行为软技能、反问环节，综合概率预测
- **风险系数建模**：核心技术卡壳、离职原因矛盾、礼仪缺失，扣减叠加
- **黄金话术生成**：STAR+ 模板，贴合实际背景
- **ReviewQA 知识库**：以问题为核心追踪历史表现

## 快速开始

### 前置准备

需要用户提供：
1. 公司名称 & 岗位名称
2. 岗位 JD（必须包含工作年限要求）
3. 面试轮次（技术面/架构面/HR面）
4. 面试录音转写文本（全文粘贴）

### 使用流程

**阶段一：解析与确认**
- 粘贴背景信息和转写文本
- 系统输出带编号的确认卡
- 用户核对、修正（可触发检查点自检）
- 回复"确认"进入诊断

**阶段二：深度诊断**
- 自动职级识别 + 五维评分
- 输出完整诊断报告（自动保存 Markdown）
- 下一轮冲刺建议 + 下轮追问预测
- 报告后邀请反馈（触发检查点2评审）

**阶段三：ReviewQA 更新**
- 自动追加 Q&A 到知识库
- 问题归并 + 评分统计

## 评分体系

### 职级模型

| 职级 | 判定标准 | 核心考核点 |
|------|---------|-----------|
| L1: 潜力者 | N < 3年 | 基础知识准确度、逻辑潜力、学习态度 |
| L2: 实战派 | 3 <= N < 8年 | 复杂问题解决能力、方案选型逻辑、业务深度 |
| L3: 布道者 | N >= 8年 | 行业视野、方法论沉淀、组织影响力与协作 |

### 权重矩阵

| 职级 | 面试类型 | W1(Intro) | W2(QA) | W3(Soft) | W4(Reverse) |
|------|---------|-----------|--------|----------|-------------|
| L2 | 技术一面/二面 | 0.15 | 0.60 | 0.15 | 0.10 |
| L2 | 架构面/系统设计面 | 0.10 | 0.65 | 0.10 | 0.15 |
| L3 | 架构面/系统设计面 | 0.10 | 0.50 | 0.10 | 0.30 |

### 概率映射

| 加权总分 | 通过概率 |
|---------|---------|
| 85+ | 90%+ |
| 75-84 | 70-89% |
| 65-74 | 50-69% |
| 55-64 | 30-49% |
| 55以下 | <30% |

## 输出文件

- 诊断报告：`interview-review-outputs/interview-review-{YYYYMMDD}-{HHMMSS}.md`
- 问题知识库：`interview-review-outputs/ReviewQA.md`

## 示例

```
用户粘贴：30分钟技术面转写文本
系统输出：阶段一确认卡 → 用户确认 → 阶段二诊断报告 → ReviewQA更新
```

## 安装

将 `interview-review` skill 目录放入 Claude Code skills 目录即可。
